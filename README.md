# POOGames

Bem vindo a POOGames! sua plataforma para cátalogar e organizar sua coleção de jogos!!

Esse sistema tem por objetivo cadastrar os seus jogos digitais, acompanhar o progresso de horas jogadas que o usuário realiza nos mesmos, avaliar os jogos e criar listas dentro da sua coleção baseado em filtros personalizados. 

As principais classes que usaremos nesse sistema serão: 
Coleção: a classe que contém todos os jogos adicionados pelo usuário.
Jogos: a classe que receberá os atributos pertinentes aos jogos.
Usuário: a classe que possui os atributos dos usuários do sistema.
Avaliação: a classe que resultará no método na qual o usuário poderá realizar as avaliações dos jogos que desejar.
Relatório: classe que receberá a data que o relatório foi gerado e os filtros que o usuário fez para gerar aquele realatório
